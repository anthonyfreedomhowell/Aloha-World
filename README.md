# Aloha-World
This is a simple Aloha world project.
This program prints "Aloha World!".

To compile this Java program use javac,versinon , 1.8.0_181. Then run the AlohaWorld.class file
with java, version 1.8.0_181. 

To compile type this into the terminal:

	javac AlohaWorld.java

Then to run the program just type:

	java AlohaWorld

The java command does not need to add the .class extension in the program argument.
